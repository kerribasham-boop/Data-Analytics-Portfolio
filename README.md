# AI Data Annotation Project

A data annotation project created in Google Sheets.

## Project Overview
This project demonstrates foundational data preparation and annotation skills used in AI and analytics workflows. Tasks include text classification, named entity recognition, transcription, and image annotation.

Each dataset includes labeled examples and notes explaining classification decisions.

---

## Annotation Tasks

| Task | Samples | Data Source | Notes |
|------|--------|------------|------|
| Sentiment Classification | 20 | Manually created sentences | Demonstrates sentiment labeling and reasoning |
| Topic Classification | 20 | Manually created health sentences, topics, and definitions | Demonstrates categorization and consistency |
| Named Entity Recognition (NER) | 20 | Transcript from royalty-free video (Pixabay) | Demonstrates entity labeling using NER guidelines |
| Image Annotation | 20 | Transcript from royalty-free image (Pixabay) | Demonstrates entity identification in visual data |
